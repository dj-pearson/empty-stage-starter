# App Icon Required

**File:** `public/icon-512x512.png`

**Dimensions:** 512x512px (square)

**Design Requirements:**
- Simple, recognizable design
- Works at all sizes (512px down to 16px)
- No text (or minimal, very bold text)
- High contrast
- Brand colors
- Transparent or solid background

**iOS Requirements:**
- No transparency (will be masked)
- Rounded corners applied automatically
- Should look good at 1024x1024px (scale up for App Store)

**Android Requirements:**
- Can have transparency
- Adaptive icon: foreground + background layers
- Safe zone: Keep important elements in center 66% (330px circle)

**Quick Generation:**
```bash
# Option 1: Use existing favicon and scale up
convert public/favicon.ico -resize 512x512 public/icon-512x512.png

# Option 2: Use Expo's asset generator
npx expo-asset-generator --icon ./design/icon-source.png

# Option 3: Design in Figma/Canva at 512x512px
```
